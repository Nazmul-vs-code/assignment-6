🛠️ DigiTools - Premium Marketplace
📝 Description
DigiTools is a sleek, modern web application designed for creators and professionals to discover and manage premium digital assets. Featuring a dynamic product catalog and a real-time shopping cart system, it provides a seamless user experience for browsing high-quality software, templates, and AI-driven resources.

🚀 Technologies Used
React.js – Component-based UI architecture.

Tailwind CSS – Modern utility-first styling for a responsive design.

DaisyUI – Accessible and beautiful UI components.

React Icons – For clean, scalable iconography.

React-Toastify – Real-time notifications for user actions.

✨ Key Features
Interactive Product Catalog: Browse a wide range of digital tools with real-time filtering between available products and your selected cart. Each product card features detailed descriptions, pricing tiers, and unique feature lists.

Dynamic Cart Management: A fully functional shopping cart that allows users to add items, remove specific products, and clear the entire selection. The system automatically calculates the Total Price using the cumulative value of all selected assets.

Premium Pricing Tiers: A beautifully designed, three-tier pricing section (Starter, Pro, and Enterprise) that clearly outlines service levels. The "Pro" tier features a high-impact linear gradient background to highlight the best-value option.
